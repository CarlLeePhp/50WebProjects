08 Form Wave Animation

When a feild is focused, its label will go up letter by letter.

The split() method and join() method are used to split the label and reconect it. Another important method is map(). It receives a function as the first parameter. The first two parameters of the function are the current value and the index. The map() method will return a new array which consist of the values retured from the function.
